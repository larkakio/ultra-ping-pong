# Assets Required

You need to create the following assets for the game:

## Required Assets

### 1. `/public/icon.png`
- **Size**: 512x512 pixels
- **Format**: PNG
- **Style**: Cyberpunk/Neon aesthetic
- **Design**: Should represent the Pong game with cyberpunk elements (neon colors, futuristic design)
- **Usage**: App icon, PWA icon, Farcaster manifest icon

### 2. `/public/hero-image.svg` or `/public/hero-image.png`
- **Size**: 1200x630 pixels (recommended for social sharing)
- **Format**: SVG (preferred) or PNG
- **Style**: Cyberpunk/Neon aesthetic matching the game
- **Design**: Hero image showing the game with "Ultra Ping Pong" branding, neon colors, cyberpunk grid background
- **Usage**: Social media preview, Farcaster manifest splash screen

## Design Guidelines

### Color Palette
- Primary Neon Cyan: `#00F0FF`
- Secondary Neon Magenta: `#FF00E5`
- Accent Yellow: `#FFFC00`
- Background: `#0A0E27`
- Dark elements: `#050505`

### Design Elements
- Neon glow effects
- Cyberpunk grid patterns
- Futuristic typography (Orbitron font)
- Retro pixel elements
- Motion/trail effects

## Quick Asset Creation Tips

1. Use design tools like Figma, Adobe Illustrator, or Canva
2. For the icon, create a simple but recognizable Pong paddle and ball design with neon effects
3. For the hero image, create a landscape composition showing the game interface or concept
4. Export at high resolution for best quality
5. Optimize file sizes without losing quality (use tools like TinyPNG)

## Temporary Placeholders

Until you create the actual assets, you can:
- Use a simple colored square as placeholder for icon.png
- Create a basic SVG with the game title for hero-image.svg

The app will work without these assets, but they're required for proper Base App and Farcaster integration.
